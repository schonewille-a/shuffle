# **Usage:**

**Production build:**
Go to **https://goo.gl/LkJvGE** for the production build

**Development build:**
Requires latest nodejs build (https://nodejs.org/en/)

 1. Open command line
 2. Navigate to Shuffle/app
 3. Run "npm install"
 4. Run "npm start"

**The app should start in your browser automatically at localhost:3000**
